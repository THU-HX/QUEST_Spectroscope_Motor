## 3. 电机控制指令列表

#### 电机1 调焦结构 x轴

| 变量            | 含义                              | 单位                     |
| --------------- | --------------------------------- | ------------------------ |
| Motor[1].ActPos | 编码器实测位置（actual position） | 光栅尺（传感器）实际返回 |
| Motor[1].ActVel | 编码器实测速度                    | 计数 / 伺服周期          |

​	

| 变量               | 含义                       | 正常值              |
| ------------------ | -------------------------- | ------------------- |
| Motor[1].AmpEna    | 电机使能状态               | 0=未使能 / 1=已使能 |
| Motor[1].AmpFault  | 电机故障                   | 0                   |
| Motor[1].SoftLimit | 限位是否触发（不区分正负） | 0                   |

#### 

| 变量   | 含义                                        |
| ------ | ------------------------------------------- |
| #1j/   | 电机使能                                    |
| #1j=xx | 控制点击移动到xx位置 绝对位置               |
| #1j:xx | 控制点击移动xx位置 相对位置                 |
| #1k    | 电机去使能                                  |
| #1j+   | 电机持续正向移动（1-4 需谨慎 光栅尺范围小） |
| #1j-   | 电机持续反向移动（1-4 需谨慎 光栅尺范围小） |
